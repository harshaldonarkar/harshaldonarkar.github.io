# Personal Portfolio

This repository hosts the source code for my personal portfolio website. It is built with static HTML, CSS, and JavaScript and deployed to GitHub Pages.

## Project Structure
- `index.html` – Main landing page.
- `styles/` – CSS files.
- `scripts/` – JavaScript utilities.

## Building & Deployment
The site is purely static, so you can simply push to the `main` branch and GitHub Pages will automatically publish it.
