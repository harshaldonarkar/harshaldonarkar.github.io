# Contributing

## This is a personal portfolio — not an open-source project

Thank you for your interest. This repository exists for version control
and public visibility, not as a collaborative project.

**Pull requests will not be accepted.** This is a bespoke portfolio
built specifically for Harshal Naresh Donarkar and is not intended to
evolve through community contributions.

---

## What you CAN do

- ⭐ **Star the repo** if you found it inspiring
- 🐛 **Open a bug report** if something is broken (see below)
- 💬 **Reach out directly** if you want to collaborate on something new

---

## Reporting Bugs

If you notice a genuine rendering issue, broken link, or browser
compatibility problem, open an issue with:

- **Browser + version** (e.g. Chrome 123, Safari 17)
- **Operating system**
- **Screenshot or screen recording**
- **Steps to reproduce**

Please do NOT open issues to request features, design changes, or
ask for permission to use the code.
